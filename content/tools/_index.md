## [TokenFlare](https://github.com/JumpsecLabs/TokenFlare)
Serverless Adversary-in-the-Middle (AiTM) phishing framework for Entra ID / M365. Deploys to CloudFlare Workers in under 60 seconds.

## [TokenSmith](https://github.com/JumpsecLabs/TokenSmith)
Entra ID authentication utility for token manipulation. Includes Intune compliant device bypass for Conditional Access Policy circumvention.

## [AHHHZURE](https://github.com/gladstomych/AHHHZURE)
Automated deployment script that creates a vulnerable Azure cloud lab for practicing offensive cloud security techniques.
